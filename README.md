# Mentally

Mentally est une application mobile développée en Flutter. 
Elle est pensée pour aider les utilisateurs à suivre leur humeur, pratiquer des exercices de relaxation et recevoir des notifications personnalisées afin de suivre leur bien-être mental.


## Fonctionnalités principales

### Journal d'humeur

- Enregistrer son humeur du jour,
- Ajouter une note,
- Visualiser son historique d'humeurs via le calendrier.
